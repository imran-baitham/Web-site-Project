Animated navigation in html css Javascript

Most of the developers use Animated navigation in their websites to make it attractive for users and increase its beauty



In this article , i will show you how to use linear gradient active when focus on button using html, css and javascript . it is easy to implement in your code


Approch to create Animated navigation using html css javascript:



    in html section you just have to create a Animated navigation with according to your requirements . In css section add focus using linear gradient make it easy using javacript.



HTML Code

  <!DOCTYPE html>
  <html lang="en">
  <head>
      <meta charset="UTF-8">
      <meta http-equiv="X-UA-Compatible" content="IE=edge">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>Animated_Navigation</title>
      <link rel="stylesheet" href="styles.css">
  </head>
  <body>
      <nav class="active" id="nav">
          <ul>
              <li><a href="#">Home</a></li>
              <li><a href="#">Works</a></li>
              <li><a href="#">About</a></li>
              <li><a href="#">Contact</a></li>
          </ul>
  
          <button class="icon" id="toggle">
              <div class="line line1"></div>
              <div class="line line2"></div>
          </button>
      </nav>
      <script src="app.js"></script>
  </body>
  </html>
